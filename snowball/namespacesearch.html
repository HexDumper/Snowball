getdir() Scans the current working directory, gets the names of the files contained within it, opens and writes into those files.

Parameters
dir: a string corresponding to the directory to be searched; files : a vector<string> corresponding to the array of the names of the files contained within the scanned directory.
Returns
Technically returns an integer, but the "files" parameter can be used like just like an ordinary array.
